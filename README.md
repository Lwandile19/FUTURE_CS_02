# FUTURE_CS_02
Public Cyber Security Internship Program at Future Interns.
# Overview
This project was completed as part of Future Interns – Cyber Security Task 2. The objective was to analyse phishing email samples, identify common phishing indicators, classify their level of risk, and develop user awareness recommendations to help prevent phishing attacks.

# Tools Used
--Google Admin Toolbox Message Header Analyzer – Analysed email headers, including SPF, DKIM, and DMARC authentication results.
--Notepad – Opened and reviewed .eml email files.
--Web Browser – Inspected sender domains and hyperlinks without interacting with malicious content.
--Microsoft Word – Documented findings and produced the final report.

# Analysis Approach
1. Collected publicly available phishing email samples in .eml format.
2. Extracted and analysed email headers using the Google Admin Toolbox Message Header Analyzer.
3. Examined sender addresses, domains, subjects, and email content for phishing indicators.
4. Identified social engineering techniques and email authentication results.
5. Classified each email according to its level of risk.
6. Documented findings and developed phishing prevention and employee awareness guidelines.
